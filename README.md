# Siba's PC Benchmark (SPCBM)

A cyberpunk-themed PC benchmarking and showcase website built with HTML, CSS and Vanilla JavaScript.

## Overview
SPCBM combines benchmark data, gameplay galleries and a premium PC showcase into an interactive experience.

## Features
- Gameplay gallery
- Premium Gaming & Creator Workstation
- Performance Highlights
- Verified Listing
- Dynamic PC component loading
- Responsive cyberpunk UI

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Lucide Icons
- GitHub Pages
